03 Variables & Data Types

A variable is a named container that stores a value in memory. In C++, every variable must have a type declared before it can be used.

💡 Think of a variable as a labelled box — the type tells you what kind of thing fits inside it, and the name is the label you use to find it later.
TypeSizeExample valueUse for
int 4 bytes42, -7 Whole numbers
float 4 bytes3.14 Decimal numbers (less precise)
double 8 bytes3.14159265 Decimal numbers (more precise)
char 1 byte 'A', 'z' A single character
bool 1 byte true, false Yes/No conditions
string varies "hello" Text (from <string> library)
C++ 
// Declaring variables
int    age    = 20;
double gpa    = 3.76;
char   grade  = 'A';
bool   passed = true;
string name   = "Zunaira";

// Printing variables
cout << "Name: " << name << endl;
cout << "GPA: "  << gpa  << endl;

04 Operators

Operators are symbols that perform operations on variables and values.

Arithmetic Operators

OperatorNameExampleResult
+Addition 5 + 38

05 Control Flow

Control flow lets your program make decisions. The most common structure is if / else if / else.

C++ 
int marks = 75;

if (marks >= 90) {
    cout << "Grade: A";
} else if (marks >= 75) {
    cout << "Grade: B";
} else if (marks >= 60) {
    cout << "Grade: C";
} else {
    cout << "Grade: F";
}

Switch Statement

Use switch when you're comparing one variable against many fixed values — it's cleaner than a long chain of else if.

C++ 
int day = 3;

switch (day) {
    case 1:  cout << "Monday";    break;
    case 2:  cout << "Tuesday";   break;
    case 3:  cout << "Wednesday"; break;
    default: cout << "Other";
}

06 Loops

Loops repeat a block of code. C++ has three types:

for loop

Best when you know exactly how many times to iterate.

C++ 
// Print 1 to 5
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
// Output: 1 2 3 4 5

while loop

Runs while a condition is true — useful when you don't know the count ahead of time.

C++ 
int n = 1;
while (n <= 5) {
    cout << n << " ";
    n++;
}
✅ Quick rule: Use for when count is known, while when condition-based, and do-while when the loop must execute at least once (like menu repetition).

07 Functions

A function is a reusable block of code that performs a specific task. Functions reduce repetition and make programs easier to read and debug.

Anatomy: returnType functionName(parameters) { body }

C++ 
// Function definition
int add(int a, int b) {
    return a + b;
}

// Function call
int result = add(5, 3);
cout << result;  // Output: 8

void Functions

If a function doesn't return a value, use void as the return type.

C++ 
void greet(string name) {
    cout << "Hello, " << name << "!" << endl;
}

greet("Zunaira");  // Output: Hello, Zunaira!

08 Arrays

An array stores multiple values of the same type under one variable name, accessed by index (starting from 0).

C++ 
// Declare and initialize an array
int marks[5] = {85, 90, 78, 92, 88};

// Access elements (index starts at 0)
cout << marks[0];  // 85
cout << marks[4];  // 88

// Loop through array
for (int i = 0; i < 5; i++) {
    cout << marks[i] << " ";
}
⚠️ Accessing an index outside the array bounds (e.g., marks[10] on a size-5 array) causes undefined behavior in C++. Always stay within range.
